The class work with .jpg, .png and .gif files.

Usage in HTML code:

// with a custom size
<img src="class_ethumb.php?image=xxxxx.jpg&size=150"/>

// with default size of 100px
<img src="class_ethumb.php?image=xxxxx.jpg"/>

// if you want to change the default size,
// go to class_etumb.php and change the line
// private $defaultSize = 100;
// to your own.
